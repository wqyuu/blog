Zookeeper入门—ZK命令行【ACL权限控制（一）】

2019-04-13 zookeeper

常用命令行

acl命令

world:anyone:cdrwa

解读：

  • c：创建
  • d：删除
  • r：读取
  • w：写入
  • a：admin，可以设置权限

命令

  • setAcl 设置节点权限
  • getAcl 获取节点权限

操作步骤
1.创建节点
create /a/b aa
2.查看权限
getAcl /a/b
‘world’,anyone: cdrwa --默认权限
......
3.设置权限
setAcl /a/b world:anyone:crwa
getAcl /a/b
'world',anyone:crwa
4.测试删除权限
create /a/b/c 1
delete /a/b/c
Authentication is not valid : /a/b/c --授权错误
...
